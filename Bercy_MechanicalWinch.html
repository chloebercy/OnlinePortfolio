Name : 1.2.5 Mechanical Winch System Project

Course : Principles Of Engineering

Date : October 24, 2017

In this project, the objective was to build a mechanical winch prototype, a simluation using Netlogo, documentation and calculations. The project was graded using this rubric. In this project, I mainly focused on the winch simulation. I created the function to set up the "environment" and briefly worked on the function that made the winch run. The main challenge for me during this project would be adjusting to the Netlogo syntax. I resolved this issue by paying more attention to the code I wrote and testing more frequently to catch mistakes.
