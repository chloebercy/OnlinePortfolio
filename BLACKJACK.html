CHLOE BERCY


Name : BlackJack Project

Course : AP Computer Science Applications

Date : May 13, 2019

In this project, I created a blackjack game with a partner using Java. The game was a singleplayer game with a computer dealer. The work was split evenly where I focused more of my time on the logic behind allowing an ace to have a value of 1 or 11 and understanding/utlizing parts of the JSwing package (such as JFrame and JPanel class) to create graphical user interface. The biggest problem faced was during the program design, rather than the program implementation. It took some time to find a solution and break it down into identifying the classes, identifying the behaviours for each class, and determining the relationship between the classes. To see our work, check out this repository.
